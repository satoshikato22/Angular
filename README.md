# Angular
## Angular CLIのインストール
$ npm install -g @angular/cli
## rootフォルダまで移動して、npmインストールを行います。
$ npm install
#アプリケーション実行方法 
##環境 Windows11Pro コマンドプロンプトにて以下を実行
## JSONサーバーを起動 http://localhost:3000/locations
$ json-server --watch db.json
## Angular実行 http://localhost:4200/
$ ng serve

# 基本的なコマンド
## ng new	プロジェクト作成
## ng serve	開発サーバー起動
## ng generate	コンポーネントなどの生成
## ng build	本番ビルド
## ng add	ライブラリ追加

# Angular
## Angular CLIのインストール
$ npm install -g @angular/cli
## rootフォルダまで移動して、npmインストールを行います。
$ npm install
## JSONサーバーを起動 http://localhost:3000/locations
$ json-server --watch db.json
## Angular実行 http://localhost:4200/
$ ng serve

# 基本的なコマンド
### プロジェクト作成
$ ng new
### 開発サーバー起動
$ ng serve
### コンポーネントなどの生成
$ ng generate
### 本番ビルド
$ ng build
### ライブラリ追加
$ ng add
